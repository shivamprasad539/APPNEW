  Online Music Website
------------------------
 
  About the Project
----------------------- 
The Online Music Website is a web-based platform that allows users to browse, play, and manage music online.
Users can explore different songs, create playlists, and enjoy a seamless music streaming experience.

This project focuses on providing an interactive and user-friendly interface for music lovers.

  Technologies Used
-----------------------
Frontend Technologies
    -> HTML
    -> CSS
    -> Bootstrap
Backend
    -> MySQL
Server-Side Scripting
    -> PHP
Client-Side Scripting
    -> JavaScript
    -> AngularJS

  Features
-------------
-> Play songs online
-> Search music by name or category
-> Create and manage playlists
-> User registration and login system
-> Store user data and preferences
